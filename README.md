## Sobre este projeto

Este repositório contém o material completo de um livro didático sobre Git e GitHub escrito em português brasileiro — do zero absoluto até o nível que as melhores empresas esperam de um desenvolvedor.

O objetivo não é decorar comandos. É entender o que o Git está fazendo por baixo dos panos — para que, quando algo der errado (e vai dar), você saiba o que fazer.

O conteúdo foi organizado como uma progressão natural: cada parte foi construída sobre a anterior. Quem vai estudar do zero lê na ordem. Quem já sabe o básico pode ir direto para a parte de referência rápida.

## O que você vai aprender

**Parte 0 — Configuração**
Instalar Git, configurar identidade, integrar com VS Code e Git Bash. Feito uma vez, usado por anos.

**Parte 1 — Fundamentos**
O ciclo básico do Git: `init`, `add`, `commit`, `push`. Mensagens profissionais com Conventional Commits. `.gitignore`. Conexão com o GitHub.

**Parte 2 — Colaboração**
Branches como linhas do tempo independentes. Merge, conflitos e como resolvê-los. Pull Requests e o GitHub Flow. Fork e contribuição em projetos open source.

**Parte 3 — Nível Profissional**
`git restore`, `git reset` (com analogia para entender `--soft`, `--mixed` e `--hard`), `git revert`. Stash para guardar trabalho temporário. Reflog para recuperar o que parecia perdido. Interactive rebase e cherry-pick.

**Parte 4 — Automação**
GitHub Actions na prática: workflows em YAML, CI com testes automáticos, deploy contínuo. Secrets, Branch Protection Rules, Dependabot, CODEOWNERS.

**Parte 5 — Referência Rápida**
Cheat sheet completo por categoria. Dicionário de problemas com soluções diretas. 11 exercícios práticos com enunciados separados das soluções.

## Como usar

**Estudando do zero:** leia na ordem, de `00` até `10`. Faça os exercícios antes de ver as soluções. Cada capítulo tem blocos `🧪 Experimente agora` — não pule. O Git só entra na cabeça quando as mãos trabalham.

**Consultando rápido:** vá direto para [`cheatsheet-git.md`](./cheatsheet-git.md) ou para a seção de dicionário de problemas em [`08-Referencia_Rapida.md`](./08-Referencia_Rapida.md).

**Travou num exercício:** tente por pelo menos 15 minutos antes de abrir [`10-Apendice_D_Solucoes_Detalhadas.md`](./10-Apendice_D_Solucoes_Detalhadas.md). Ler a solução sem tentar não ensina nada.

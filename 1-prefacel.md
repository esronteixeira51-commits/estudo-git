
## Prefácio

**Git & GitHub — Do zero ao nível profissional**

Imagine que você está construindo uma casa.

Você não começa colocando o telhado. Você começa pelos alicerces. Se o alicerce for mal feito, mais tarde tudo racha, vaza ou desaba.

**Git é o alicerce do desenvolvimento de software moderno.**

Sem ele, trabalhar sozinho já é confuso. Trabalhar em equipe vira um pesadelo: alguém sobrescreve o código do outro, mudanças somem, ninguém sabe quem quebrou o quê, e voltar para uma versão que funcionava vira quase impossível.

Com Git bem feito, tudo muda. Você pode trabalhar em novas funcionalidades sem medo de estragar o que já está bom. Pode colaborar com outras pessoas — ou com você mesmo de amanhã — sem brigas. Pode voltar no tempo quando algo dá errado. E, principalmente, pode mostrar seu trabalho de forma profissional para recrutadores, clientes e times de desenvolvimento.

Este livro nasceu exatamente dessa ideia simples: **ensinar Git e GitHub de forma que você realmente entenda, não apenas decore comandos.**

Aqui está reunido todo o material que foi usado para ensinar dezenas de alunos — desde quem nunca tinha visto um terminal até quem já trabalhava em empresa mas ainda tinha medo de fazer `git rebase`. O conteúdo foi organizado em uma progressão natural: do zero até o nível que as melhores empresas esperam de um desenvolvedor.

Você vai encontrar explicações claras, analogias que fazem sentido, muitos exercícios práticos reais e, no final, uma referência rápida para consultar quando precisar.

**O objetivo é um só:**

Quando você terminar este livro, Git deixe de ser "aquela coisa complicada" e passe a ser uma ferramenta natural — como dirigir um carro ou usar o celular.

Porque, no final das contas, Git não é sobre decorar comandos. É sobre entender o que está acontecendo por baixo dos panos. Quando você entende, o medo desaparece e a confiança aparece.

Bem-vindo à jornada. Vamos construir um alicerce sólido juntos.
